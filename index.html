Welcome to Bridges, the innovative and challenging puzzle game!

Also known as Hashiwokakero, the objective of Bridges is to connect all the islands with the right number of bridges to complete each puzzle. Bridges connect islands vertically and horizontally, depending on how many bridges each island needs. Complete each puzzle by creating one fully connected network - no crossovers! Stimulating and entertaining, Bridges is a game that is easy to learn, hard to master, and always fun to play!

New levels will be added regularly, so make sure to bookmark this page and check it often!
